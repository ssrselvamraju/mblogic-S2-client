HMI Messages

Overview:

HMI Message display shows the actual messages which are transmitted between the client (web browser) and the HMI server. This is mainly of use when creating a new web page.

The request column shows the messages from the web client (browser) to the server. The response column shows the replies from the server to the web client.

The message cycle is driven by the client. The server responds to requests from the client, but does not initiate communications on its own.
